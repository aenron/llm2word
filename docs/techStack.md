## 后端框架
- FastAPI：提供 HTTP 接口，接收外部 JSON 并返回 DOCX 二进制文件。

## 文档生成
- docxtpl：以 Word 模板为基础进行变量渲染。
- python-docx：用于模板引导创建与段落级样式控制（行距、缩进、字体等）。

## 数据建模
- Pydantic v2：定义请求体结构与样式参数约束。

## 运行环境
- Python 3.10+（推荐 3.11）。
- Uvicorn：本地开发启动服务。

## 架构决策
- 采用“外部 JSON + 服务端渲染 + 文件流返回”模式，避免让大模型直接生成二进制 docx。
- 采用模板引导创建机制，保证项目首次启动可运行，同时保留后续人工精调模板能力。
